## zephyr
The zephyr is a symbol of hope in spring
## 快速启动
 启动注册中心：进入注册中心目录，zephyr-registor/nacos/bin 执行命令startup -m standalone 单机模式启动注册中心
 
 默认端口8848  默认用户名:nacos  密码:nacos 
## 端口说明
网关 9000
认证中心 10010
系统服务 10020
